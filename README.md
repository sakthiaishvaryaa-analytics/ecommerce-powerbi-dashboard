# ecommerce-powerbi-dashboard
#E-Commerce Sales Performance Dashboard built using Power BI, DAX and Superstore dataset
An interactive sales analytics dashboard built using Power BI,
analysing Superstore e-commerce data across regions, categories,
and products.

## Tools Used
- Power BI Desktop
- DAX (Data Analysis Expressions)
- Power Query (M Language)
- Superstore Sales Dataset

## Key Metrics
- Total Sales: $1.57M
- Total Profit: $175.26K
- Profit Margin: 11.19%

## Key Insights
1. West region leads in both Sales and Profit (38.72%)
2. Technology category drives highest sales at 41.11%
3. Copiers is the most profitable sub-category
4. November–December shows clear seasonal sales spike
5. Overall profit margin is healthy at 11.19%

## Dashboard Visuals
- KPI Cards — Total Sales, Profit, Profit Margin %
- Sales by Region (Bar Chart)
- Sales by Month (Line Chart)
- Sales by Category (Donut Chart)
- Top Products by Sales (Horizontal Bar)
- Profit by Sub-Category (Bar Chart)
- Profit by Region (Donut Chart)
- Interactive Slicers — Region, Category
